Выражения и операторы

зачем выражения и операторы

Выражения

Выражения в языке это участок кода, который вернет значение интерпретатору
Например:

  • "string"
  • 123

lalala

Операторы

Важно понимать, что все операторы имеют приоритет, приоритет определяет в какой последовательности будут вычисляться выражения. Влиять на приоритет операторов можно при помощи скобок.

Операторы присваивания

Знак оператора присваивания это =, когда происходит присваивание, значение правого операнда присваивается левому операнду
a - левый операнд, b - правый операнд, = - оператор присваивания

a = b

Список операторов присваивания

Имя Сокращёние Смысл
Присваивание x = y x = y
Присваивание со сложением x += y x = x + y
Присваивание с вычитанием x -= y x = x - y
Присваивание с умножением x *= y x = x * y
Присваивание с делением x /= y x = x / y
Присваивание по модулю x %= y x = x % y
Присваивание с левым сдвигом x <<= y x = x << y
Присваивание с правым сдвигом x >>= y x = x >> y
Присваивание с беззнаковым сдвигом вправо x >>>= y x = x >>> y
Присваивание с побитовым AND x &= y x = x & y
Присваивание с побитовым XOR x ^= y x = x ^ y
Присваивание с побитовым OR x |= y x = x | y

Деструктуризация

Рассмотрим еще один способ присваивания, данный способ отлично подойдет для взаимодействия с массивами и объектами

var foo = ["one", "two", "three"];
// без деструктуризации
var one = foo[0];
var two = foo[1];
var three = foo[2];
// с деструктуризацией
var [one, two, three] = foo;

Операторы сравнения

Оператор сравнивает операнды и возвращает логическое значение true или false. Тип данных операнд может быть различных типов.

Обьявим переменную a со значением 1 и переменную f со значением 3
Пример:
let a = 1
let f = 3

Оператор Пример Результат
Равно == a == f false
Не равно != a != f true
Строго равно === a === f false
Строго не равно !== a !== f true
Больше > a > f false
Больше или равно >= a >= f false
Меньше < a < f true
Меньше или равно <= a <= f true

Арифметические операторы

Операндом в данном случае могут выступать и числа
Пример:
1 + 3
let a = 1
let f = 3
a + f
Результатом выполнения всегда одно значение, с типамом данных число

Оператор Описание Пример
+ добавление 1 + 3
- вычитание 3 - 1
* умножение 2 * 2
** возведение в степень (ES 2016) 2**2
/ деление 4 / 2
% модуль (остаток) 3%2
++ инкремент ++a, a++
-- дикремент --a, a--

Инкремент и декремент прменяются только к переменным, их два вида постфиксный и префиксный.
Постфиксный a++ или a-- сначала возвращает текущее значение, потом изменяет значение переменной.
Префиксный ++a или --a сначала изменяет значение, потом выводит уже измененное значение


Логические операторы

Логические выражения вычисляются слева направо, логические операторы оперируют логическими значениями true и false но значения возвращаются текущие, заданные в операнд.

Оператор Пример Описание
Логическое И - && operand1 && operand2 вернется true если обе операнды true, иначе вернется false
Логическое ИЛИ - || operand1 || operand2 Если один из операнд после преобразования будет true то true и вернется, если оба false вернется false
Логическое НЕ - ! !operand1 Возвращает false, если операнд может быть преобразован в true, иначе вернется true.

В false преобразуются - null, 0, NaN, пустая строка ("") или undefined.

Пример c И && Результат
true && true true
true && false false
false && true false
false && (3 == 4) false
"Cat" && "Dog" Dog
false && "Cat" false
"Cat" && false false
Пример с ИЛИ || Результат
true || true true
false || true true
true || false true
false || (3 == 4) false
"Cat" || "Dog" Cat
false || "Cat" Cat
"Cat" || false Cat
Пример с НЕ ! Результат
!true false
!false true
!"Cat" false

Строковые операторы

При работе со строками, их можно складывать, при помощи оператора +, данный оператор обьединяет строки а процесс называется конкатинацией. В языке JavaScript можно сложить два типа данных - число и строку, в результате получится строка.

Пример Результат
"Hello" + "World" "HelloWorld"
"Hello World" + 243 "Hello World243"

Тернарный оператор

условие ? value1 : value2

Если условие вернет true то выполняется первое выражение value1, если условие false то выполнится второе выражение value2


Пример:

true ? console.log('Value1') : console.log('Value2')


Оператор запятая

Данный оператор выполняет оба операнда слева направо и возвращает значение второго операнда. Оператор запятая имеет наименьший приоритет выполнения.
Пример:
let a = (1, 3)
переменной а присвоится значение 3


Оператор typeof

Используя оператор typeof можно узнать какой тип операнда
Пример:
typeof operand или typeof(operand)

Пример Результат
typeof 1 number
typeof 'string' string
typeof true boolean
typeof undefined undefined
typeof {} object
typeof null object
typeof function function
