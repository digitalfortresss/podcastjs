Типы данных

типы данных в js

Примером ранее, мы вывели строку "Hello World" в консоль разработчика, а сейчас давайте выучим типы данных которые присутствуют в js. Очень важно знать какие типы данных есть в языке.

  • string - строка
  • number - число
  • bigInt - Целые числа произвольной точности
  • object - обьект
  • boolean - для true и false
  • undefined - для пустых значений
  • null - для неизвестных значений
  • symbol - символ

1. string - в строке хранятся символы, под символом имеется любое введенное значение, так же строка может быть пустой, т.е содержать 0 символов.
"" - пустая строка
"12345" - строка содержащая числовые символы, все так же является строкой

2. number - все просто, храним числа в 64 битном формате, не превышающие 2 в 53 степени
